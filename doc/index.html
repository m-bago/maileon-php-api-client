Overview

This is the API documentation for the Maileon PHP API wrapper that wraps the public REST API for easier use with PHP. You can use our service wrappers to communicate with the following resources:

  • Contact Events are manipulated using com_maileon_api_contactevents_ContactEventsService
  • Contactsare manipulated using com_maileon_api_contacts_ContactsService
  • Contact Filters are manipulated using com_maileon_api_contactfilters_ContactfiltersService
  • Mailings are manipulated using com_maileon_api_mailings_MailingsService
  • Transactions are manipulated using com_maileon_api_transactions_TransactionsService
  • The service connection monitoring service is available through com_maileon_api_utils_PingService

In addition to the services and their data structures, we provide some utility classes that help with XML serialization and communicating with RESTful web services:

  • com_maileon_api_AbstractMaileonService is a useful base class for implementing service wrappers for REST resources.
  • com_maileon_api_xml_AbstractXMLWrapper serves as a base class for any data object that can be serialized to or deserialized from XML.
  • com_maileon_api_HTTPResponseCodes helps with translating HTTP status codes.
