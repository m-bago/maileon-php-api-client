Welcome to the API-Client documentation or XQueue Maileon

This documentation describes the classes and methods of the PHP API client for the XQueue Maileon REST API.

Getting the Client

The recommended method to install the plugin is to use composer and download the recent version from Packagist.
composer require xqueue/maileon-api-client

However, you can also download the client from Github directly and deal with classloading.

Testing the code

In the future, the calls will be provided as PHP unit tests. So far, examples for most calls can be found in papapitest.php in Github.
